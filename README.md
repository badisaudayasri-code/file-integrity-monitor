


## 📌 Project Description
The File Integrity Monitor is a cybersecurity tool developed using Python that ensures the integrity of important files by monitoring unauthorized changes. It works by creating a baseline of cryptographic hash values for files and later compares them to detect modifications or deletions.

This project is commonly used in **cyber security and ethical hacking** to identify tampering, malware activity, or unauthorized access.

---

## 🎯 Objectives
- To monitor files for unauthorized changes
- To detect file modification and deletion
- To maintain file integrity using SHA-256 hashing
- To provide alerts when integrity violations occur

---

## 🛠️ Technologies Used
- Python 3
- hashlib (SHA-256)
- os
- json
